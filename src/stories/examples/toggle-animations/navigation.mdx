# `<toggle-element>`

<br />
<br />
## Getting started

`<toggle-element>` is a web component that provides toggle functionality to hide and show an element.

<br />
### Styling

The web component can be styled via classes.

```html
<toggle-element class="square"></toggle-element>
```

<toggle-element class="square" translate-x='-50px, 0' scale='1.5' duration='2'></toggle-element>

Toggle element can be used as a container for other elements.

```html
<toggle-element opacity="0,1" duration="3">
  <h3>This is an h3 inside a toggle-element tag</h3>
</toggle-element>
```

<toggle-element opacity="0,1" duration="3">
  <h3>This is an h1 inside a toggle-element</h3>
</toggle-element>

```html
<toggle-element translate-x="100px" duration="3">
  <div class="square"></div>
</toggle-element>
```

<toggle-element translate-x="100px" duration="3">
  <div class="square"></div>
</toggle-element>

## Hide/Show Functionality

### Showing the Element

The `show` attribute controls the visibility of the element. When you add the `show` attribute it will trigger the enter animation.

```html
<toggle-element show opacity="0,1"></toggle-element>
```

### Hiding the Element

To hide the element, remove the show attribute in the toggle-element tag. This will trigger the exit animation.

```html
<toggle-element opacity="0,1"></toggle-element>
```

### Hide and Show triggers

You can add triggers to show or hide the element by defining the `trigger-type`, `trigger-controls`, and `triggered-by` attributes.

#### Trigger type

The trigger-type specifies the type of event that triggers the show/hide action. Trigger type can be click or hover.

```html
<toggle-element trigger-type="click"></toggle-element>
```

#### Trigger controls

The trigger-controls attribute should contain the ID of the element that will be shown or hidden.

```html
<toggle-element trigger-controls="element-id" trigger-type="click"></toggle-element>
```

```html
<toggle-element id="element-id" opacity="1,0"></toggle-element>
```

#### Triggered by

The triggered-by attribute should contain the ID of the element that triggers the action.

```html
<toggle-element trigger-by="element-id"></toggle-element>
```

### Enter and Exit animations

The `ToggleElement` allows you to define different animations using animatable CSS properties.

#### Default animatable property

If you define an animatable property such as `opacity="1,0"` it will animate the toggle element from an opacity of 0 to an opacity of 1 when it appears.
Additionally, when you hide the toggle element, it will transition from an opacity of 1 to an opacity of 0.

```html
<toggle-element show opacity="0,1"></toggle-element>
```

#### Custom enter animation

To define a custom enter animation you can use the `out-` prefix followed by the animation property.

```html
<toggle-element show in-opacity="0,1"></toggle-element>
```

#### Custom exit animation

To define a custom exit animation you can use the `out-` prefix followed by the animation property.

```html
<toggle-element show out-opacity="1,0"></toggle-element>
```

### Multiple animations

You can define multiple enter and exit animations combining the methods above

```html
<toggle-element
  show
  opacity="0,1"
  in-translate-y="-200px, 0"
  out-translate-y="0,-100px"
></toggle-element>
```
